* Introduction

** Requirements

*** Python 3.7

*** [[https://github.com/hizzgdev/jsmind][jsmind]]

#+begin_src sh
git clone https://github.com/hizzgdev/jsmind.git

** Usage

#+begin_src sh
./org2mind.py -i mind.org -o mind.html
#+end_src
